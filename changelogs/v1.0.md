# v1.0 ‚Äî Initial Public Release

**Released:** 2026-03-28

The first public release of Budget Tracker ‚Äî a privacy-first personal finance forecasting tool that runs entirely on your machine.

---

## Highlights

- **Multi-account management** ‚Äî Track checking, savings, and more. Each account has its own balance, income, expenses, and projections.
- **Balance projection engine** ‚Äî Day-by-day simulation applying income and expenses based on 7 recurrence intervals (one-time, daily, weekly, biweekly, monthly, quarterly, yearly).
- **Inter-account transfers** ‚Äî An expense in one account automatically appears as income in another. Transfer events use a naming convention (`TransferName-Mon`) in the target account.
- **5 interactive chart views** ‚Äî Projection (area), Spending by Category (donut), Income vs Expenses (grouped bar), Cash Flow (net bar), Expense Trends (stacked area).
- **Ledger view** ‚Äî Transaction-level table with running balance, 4 filter modes, full-text search, and summary statistics.
- **Spreadsheet exchange** ‚Äî Export to professionally formatted Excel workbook (6 sheets with instructions, validation, color coding). Import modified workbook to create/update/delete records.

## All Changes

### Core Features
- Balance projection engine (`server/src/routes/projections.ts`) with `matchesInterval()` for all 7 interval types
- `Account` model with cascade relations to all data entities
- `BalanceSnapshot`, `IncomeSource`, `PlannedExpense`, `PriceAdjustment`, `CategoryColor` models
- All API routes scoped under `/api/accounts/:accountId/`
- Account CRUD with guard against deleting last account
- Income CRUD with toggle active/inactive
- Expense CRUD with category, variable pricing, transfer support
- Price adjustment management for variable expenses
- Category colors with global scope (shared across accounts)
- Seed script creates default "Primary" account on fresh database

### Client Application
- React 19 single-page app with centralized state in App.tsx
- Vite dev server with API proxy to Express backend
- DaisyUI v5 component styling with Tailwind CSS v4
- Recharts v2.15 for all chart visualizations
- Account selector dropdown with localStorage persistence
- Date range bar with presets (30d, 60d, 90d, 6mo, 1yr) and custom picker
- Collapsible Forecast Income and Forecast Expenses panels
- Inline entry forms for income and expense creation/editing
- Account management modal (create, rename, delete)
- Balance setup modal (auto-prompted on first load)
- Spreadsheet export/import controls with result toast notifications

### Infrastructure
- Docker Compose with 3 services (PostgreSQL 16, Express server, Vite client)
- Server hot reload via `tsx watch`
- Client hot reload via Vite HMR with volume mounts
- Server Dockerfile: `prisma db push` + `prisma generate` + seed on startup
- Health checks on all services
- `.env.example` with safe defaults for local development
- Makefile with 20+ commands for development workflow

### Documentation
- README with features overview, quick start, project structure, Makefile reference
- CLAUDE.md with architecture notes and API routes table
- `.gitignore` covering `.env`, `.DS_Store`, `.idea/`, `Dev-Notes/`, `*.xlsx`

### Security
- No personal data in repository (verified via security audit)
- `.env` excluded from git tracking
- Database credentials via environment variables only
- CORS open (acceptable for local-only tool)
